# Upwork Portfolio Note

## Portfolio title

Data Quality ETL Starter: Clean, Validate, and Export Messy CSV/API Data

## Portfolio description

Built a small Python data workflow starter that ingests messy CSV, Excel, JSON, and mock API data, normalizes columns, flattens nested JSON, checks expected columns, detects missing values and duplicate rows, exports cleaned CSV and database outputs, and generates a clear data quality report.

The project includes a CLI workflow, sample input/output files, pytest tests, Docker support, screenshots, and documentation. It is designed as a practical reference for small-team data cleaning, validation, reporting automation, and lightweight ETL tasks.

## Proposal reference sentence
